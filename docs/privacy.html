プライバシーポリシー

Git-PR Reviewer Pro（以下「本拡張機能」）は、お客様のプライバシー保護を最優先に考えています。 本ポリシーでは、本拡張機能が取り扱う情報の種類とその利用・保存方法についてご説明します。

1. 収集する情報

  • 本バージョン（v0.1.0）ではユーザー情報の収集は行いません。

本拡張機能では上記以外の個人情報や行動履歴なども一切収集しません。

2. 今後の予定

次期バージョン（v0.2.0）では、AI 要約機能を追加予定です。 実装時には OpenAI API キーを暗号化して保存します。

2. 情報の利用目的

保存された API キーは、本バージョン（v0.1.0）では実際には送信しませんが、 将来実装予定の AI 要約機能等で OpenAI API を呼び出す際に利用されます。

3. 情報の保存および管理

  • API キーは chrome.storage.local に暗号化して保存され、 復号鍵は sessionStorage にのみ保持されます。
  • ブラウザを閉じると sessionStorage はクリアされるため、 復号鍵は永続的に保存されません。

4. 第三者提供

本拡張機能は、収集した情報を第三者に提供、販売、共有することは一切ありません。

5. 外部リソースの利用

本拡張機能に含まれるすべてのスクリプト・スタイルシートはパッケージ内に含まれており、 外部サーバーからのコード読み込みや eval()、リモートコードの実行は行いません。

6. プライバシーポリシーの変更

本ポリシーは予告なく内容を変更する場合があります。最新の内容は本ページでご確認ください。
